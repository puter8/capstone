# Elevator Speech — Pally

영어 회화 학습자의 중도 이탈 문제를 대화 진행에 따라 성격이 변하는 AI 캐릭터와의 유대감 형성으로 해결하여 회화 학습 지속을 돕는 서비스

---

## 1. 문제와 해결법

### Problem

- 한국인 영어 학습자 수: **2,300만 명**
- AI 영어 회화 앱 3개월 이탈률: **78%**
- 핵심 원인: **개인화 부재**

**구체적 문제점**

- 이전 대화 내용·성향 기억 불가
- 매 세션마다 초면 관계 반복
- 피드백 이력 축적 없음
- 모든 사용자에게 동일한 응답 톤 제공

### Solution

**나를 기억하고, 회화에 최적화된 AI — 나에게 맞는 성격으로 진화하는 AI 영어 파트너**

- 이전 대화·성향 자동 기억 및 누적
- 세션이 쌓일수록 깊어지는 관계 모델
- 틀린 표현을 끊지 않고 자연스럽게 교정
- 피드백 히스토리 기반 맞춤 교정 누적
- 말투·유머·에너지 등 5축 실시간 분석
- 나만을 위해 진화하는 AI 페르소나

### 차별점

**차별성**

기존 AI 영어 회화 앱(Duolingo, Speak, 튜터링 등)은 모두 세션 단위로 초기화되는 구조다. 즉, 오늘 내가 "직장 상사와의 갈등"을 주제로 대화했더라도 다음 세션에서 AI는 그 사실을 모른다. Pally는 이전 대화와 성향을 누적 기억하기 때문에, 세션이 거듭될수록 AI가 나라는 사람을 구체적으로 알아간다. 이는 단순한 기능 추가가 아니라, 서비스의 근본 구조 자체가 다르다는 의미다.

**독특성**

말투·유머·에너지 등 5축 실시간 분석과 진화하는 AI 페르소나의 결합은 현재 시장에 존재하지 않는 개념이다. 기존 서비스들이 사용자를 레벨(초급/중급/고급)로만 분류한다면, Pally는 사용자의 커뮤니케이션 스타일 자체를 다차원으로 파악한다. 예컨대 "유머 반응이 높고, 에너지가 낮은 아침 시간대에 짧은 대화를 선호한다"는 식의 페르소나 모델은 어떤 기존 서비스도 구현하지 않았다.

**우수성**

이탈률 문제의 핵심 원인은 동기 지속 실패다. 언어 학습 동기 이론(Dörnyei, 2009)에 따르면 학습자가 장기적으로 지속하는 가장 강력한 동인은 "관계"와 "자율성"이다. Pally는 두 가지를 모두 충족한다. 관계는 캐릭터와의 정서적 유대감으로, 자율성은 나만을 위해 진화하는 페르소나로 충족된다.

Pally는 기존 서비스가 "학습 도구"에 머무는 것과 달리 지속적 관계 경험을 설계하며, 이것이 78% 이탈률 문제를 구조적으로 해결하는 유일한 접근 방식이다.

## 3. MVP / 배포

### 스타트 기간 내 MVP

- Rule-Based로 사용자의 대화 패턴을 분석할 수 있는 AI 모델
- 사용자의 대화 기록을 바탕으로 피드백 노트 제공이 가능한 AI 모델
- 최소 대화 작용이 가능한 UI

### 배포 방식

**프론트엔드 — Vercel**
Next.js 14 기반 앱을 Vercel에 배포한다. GitHub 저장소와 연동하여 main 브랜치에 푸시 시 자동 빌드 및 배포(CI/CD)가 이루어진다. Vercel의 엣지 네트워크를 통해 전 세계 CDN 캐싱이 적용되며, App Router의 서버 컴포넌트와 API Route도 서버리스 함수로 자동 처리된다.

**백엔드 — Railway**
FastAPI(Python) 서버를 Railway에 배포한다. Dockerfile 또는 requirements.txt 기반으로 컨테이너가 자동 빌드되며, 환경변수 관리와 로그 모니터링을 Railway 대시보드에서 통합 관리한다. 서버는 항상 켜진 상태(always-on)로 운영되어 응답 지연 없이 API 요청을 처리한다.

**DB / 인증 / 벡터 검색 — Supabase**
PostgreSQL 기반의 Supabase를 데이터 저장소이자 인증 서버로 사용한다. RLS(Row Level Security)를 적용하여 사용자별 데이터 접근을 DB 레벨에서 제어한다. pgvector 확장을 통해 대화 임베딩 벡터를 저장하고, RAG 파이프라인에서 유사 대화 검색에 활용한다.

## 4. PMF & 고객 인터뷰

**고객 1** — AI 영어학습 앱 이용 경험 있음 / 대학생 / 해외 무경험자
영어 학습 앱을 유료로 사용해봤으나 재미 없고 반복적이어서 탈퇴. 매일 반복되는 루틴과 실제로 쓰이는지 알 수 없는 표현들 때문에 고민.

**고객 2** — 교환학생을 위해 TOEFL 학원 다녔으나 회화 도움 X
공부와 회화는 별개이기 때문에 끊임없이 대화할 수 있는 상대가 필요하다고 느낌. 학원에서 큰 도움을 받지 못했으며, 회화 공부를 어떻게 해야 할지 모름.

**고객 3** — Chat GPT로 회화 경험 있음 / 영어 프리토킹 가능자
영어권 체류 후 AI로 회화 경험을 시도했으나 단답만 반복되고 맥락 이해가 안 돼 사용 중단. 프롬프팅을 시도해도 음성 모델 사용 시 맥락을 잊어버리는 문제가 지속 발생하여 중단.
