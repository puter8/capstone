# ✍️ LexiWriter (렉시라이터)
> **"당신의 상상력에 어휘의 날개를 달아주세요."**

소설을 쓰고 싶지만 매번 비슷한 단어와 부족한 문장력 때문에 막막했던 경험이 있나요?  
**LexiWriter**는 작가와 일반인들이 더 풍부하고 다채로운 문장을 구사할 수 있도록 돕는 **소설 창작 보조 AI 어시스턴트**야.

## 🚀 주요 기능
* **어휘 추천 및 확장**: 문맥에 맞는 고급 어휘나 묘사적인 표현을 실시간으로 추천해.
* **문장 스타일 변환**: 밋밋한 문장을 소설적인 문체(묘사 중심, 심리 중심 등)로 다듬어줘.
* **소재 및 배경 지식 제공**: 특정 상황(예: 19세기 런던, 판타지 세계관)에 어울리는 소재와 단어를 제안해.
* **LLM 파인튜닝 모델**: 일반적인 대화형 AI가 아닌, 실제 문학 작품 데이터를 학습하여 '작가다운' 문장을 생성하도록 튜닝됐어.

## 🛠 기술 스택
* **Model**: Fine-tuned LLM (Llama 3 또는 Mistral 기반)
* **Dataset**: 국내외 고전 및 현대 소설, 수필 데이터셋
* **Framework**: PyTorch, HuggingFace Transformers
* **Deployment**: FastAPI / Streamlit (사용자 인터페이스)

## 💡 기대 효과
* **문장력 극복**: 어휘력이 부족해 집필을 포기했던 초보 작가들이 끝까지 완결을 낼 수 있게 도와줘.
* **작품의 질 향상**: 진부한 표현에서 벗어나 독자의 몰입감을 높이는 풍부한 묘사가 가능해져.
* **집필 속도 단축**: 적절한 단어를 찾느라 멈춰있던 시간을 줄이고 창작의 흐름을 유지할 수 있어.
